# Todo List Application

This project is a simple Todo List web application developed as part of an AI-assisted development assignment.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple and clean user interface

## Technologies
- HTML
- CSS
- JavaScript
- Replit Agent
